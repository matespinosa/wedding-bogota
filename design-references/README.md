# Referencias de diseño — Versión alternativa

Set completo de referencias visuales (una imagen por sección) para la invitación
de Mateo & Julieth. Generadas con la skill `imagegen-frontend-web`, dentro del
mundo de marca botánico-editorial (verde bosque, latón ceremonial, papel).

Son mockups de dirección visual, **no** código. Sirven para explorar una versión
diferente del sitio antes de implementarla.

## Secciones

1. `ref-hero.png` — Hero (centrado sobre foto full-bleed)
2. `ref-fecha.png` — Fecha + cuenta regresiva
3. `ref-historia.png` — Nuestra historia (editorial offset)
4. `ref-lugares.png` — Lugares: ceremonia + recepción (50/50)
5. `ref-cronograma.png` — Cronograma del día (timeline)
6. `ref-rsvp.png` — Confirmación de asistencia (RSVP)

## Nota

Las imágenes pueden incluir nombres, lugares u ornamentación de ejemplo inventados
por el generador. El contenido real vive en `index.html`.
